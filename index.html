LEGO Sets Visualization

This Shiny App is for searching and visulizating LEGO Sets information.

The dataset is from Rebrickable.com that contains the basic information of each set (set id, year, number of pieces, theme, set name).

Application will allow to search Lego set using year, Theme or based on number of pieces.

  • Data Source: http://rebrickable.com/downloads

The Application

To display the understanding of using shiny to build an application, a simple application called LEGO Set Lookup Tool has been at: https://kerrmittech.shinyapps.io/LEGOSetDB/

This Shiny App is for searching and visualizing LEGO Sets from the years 1950 to 2016.

The Data

This application is based on the Rebrickable.com LEGO set database.

Dataset has been obtained from here and processed for the course project.

Source code for the project is available on my GitHub page.
